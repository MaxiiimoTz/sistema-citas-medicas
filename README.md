# Sistema Web de Gestión de Citas Médicas

Sistema web desarrollado con arquitectura cliente-servidor para la gestión de citas médicas.

## Tecnologías

Frontend:

- React
- Axios
- React Router

Backend:

- Spring Boot
- Java
- JPA / Hibernate

Base de datos:

- MySQL

## Arquitectura

Cliente:

- Aplicación Web (React)

Servidor:

- API REST (Spring Boot)

## Funcionalidades

- Registro de usuarios
- Inicio de sesión
- Gestión de citas médicas
- Panel de administrador
- Panel de paciente
